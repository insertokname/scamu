# *scam***u**

*Software Code Acting as Machine* (em)**u**(lator) is an emulator written in rust.

see the [S.C.A.M.](https://github.com/insertokname/SCAM) project as a reference.


### Prerequisites

- Install [Rust](https://rust-lang.org/learn/get-started/)

### Building

To build the project:

```bash
cargo build
```

The output binaries/libraries will be located in the `target/debug/` directory for debug builds, or `target/release/` for release builds.

### Running Tests

To run the tests:

```bash
cargo test
```
